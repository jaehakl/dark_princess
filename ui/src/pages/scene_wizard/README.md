
<좌측 탐색 영역>
- 처음에는 소설 장면 입력용 텍스트 상자 하나만 보여준다.
- 소설 장면을 입력하고 제출하면 시멘틱 검색을 통해 유사 장면들의 목록을 보여준다.
- 사용자는 여기서 제안된 장면을 선택하거나, 빈 장면부터 시작("새로 시작") 할 수 있음

<우측 편집 영역>
- 선택을 하면, 우측에 다음 정보들을 표시한다.
     (1) SEED 이미지
          - 기존 데이터 이미지
          - "새로 시작" 인 경우에는 노이즈 seed
          - noise seed 재생성 가능 (기존 데이터가 있어도 가능)
     (2) 배경, 인물, 대상, 행동, 디테일 (프롬프트 형태)
          - 기존 데이터 
          - 우측에 항목별 추천 태그들을 칩으로 표시 ( recommendPromptColumns API 사용)
          - "새로 시작" 인 경우에는 처음부터 추천 태그들로 세팅
     (3) 장면 스크립트
          - 기존 데이터 스크립트
          - "새로 시작" 인 경우에는 좌측 탐색영역에서 입력했던 소설 장면 텍스트를 가져와서 표시          

- 저장은 항상 "기존 데이터에 저장", "새로 저장" 두 종류가 가능하다. ("새로 시작" 인 경우에는 한번 "새로 저장" 하고 나면 "기존 데이터에 저장" 이 활성화된다.)


[필요한 백엔드 기능]
- 텍스트 시멘틱 검색으로 장면 목록 찾아오기
- 장면 프롬프트 타입별 보관, 타입별 임베딩 보관
- 장면 타입별 분리
