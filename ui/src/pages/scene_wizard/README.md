<작업 Flow>
- 처음에는 소설 장면 입력용 텍스트 상자 하나만 가운데 배치하여 보여준다.
- 소설 장면을 입력하고 제출하면 시멘틱 검색을 통해 유사 장면들의 목록을 보여준다.
- 사용자는 여기서 제안된 장면을 선택하거나, 빈 장면부터 시작("새로 시작") 할 수 있음
- 선택을 하면 
     (1) SEED 이미지
     (2) 배경, 인물, 대상, 행동, 디테일 (프롬프트 형태)
     (3) 장면 스크립트

핵심
- 소설 장면을 입력한다.

<가능한 액션>
- 장면에 어울리는 프롬프트를 생성한다.
- 장면에 어울리는 다른 장면들을 불러와서 선택한다.

<2차 액션>
- 다른 장면을 선택하여 그림만 가져온다.



- 이미지 생성 설정은 네비바에서 글로벌하게 적용하고 세션스토리지로 공유



[필요한 백엔드 기능]
- 텍스트 시멘틱 검색으로 장면 목록 찾아오기
- 장면 프롬프트 타입별 보관, 타입별 임베딩 보관
- 장면 타입별 분리
